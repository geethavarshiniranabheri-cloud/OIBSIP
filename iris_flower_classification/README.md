# Iris Flower Classification 

##  Project Description

This is a beginner-level Machine Learning project that predicts the species of an iris flower using flower measurements.

The model is trained using the Iris dataset from Scikit-learn and uses the K-Nearest Neighbors (KNN) algorithm for prediction.

The project classifies flowers into:
- Setosa
- Versicolor
- Virginica

based on:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

#  Technologies Used

- Python
- Scikit-learn
- NumPy
- Pandas
- Matplotlib

---

#  How the Project Works

1. Load the Iris dataset  
2. Split data into training and testing sets  
3. Train the KNN Machine Learning model  
4. Predict flower species using input measurements  
5. Check model accuracy  
6. Visualize data using graphs
